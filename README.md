# Employee Promotion Prediction Using Machine Learning

## 📌 Project Overview
This project applies machine learning techniques to predict employee promotions using HR data.  
The goal is to support data-driven HR decision-making by identifying key factors that influence employee promotion outcomes.

---

## 🎯 Problem Statement
Organizations often rely on subjective evaluation for promotions.  
This project aims to build a predictive model that identifies employees likely to be promoted based on historical HR data.

---

## 📊 Dataset
- Source: Kaggle Employee Promotion Dataset  
- Records: 100,000 employees  
- Features: 43 HR-related attributes  
- Target: Promotion status (0 = Not Promoted, 1 = Promoted)

---

## ⚙️ Workflow
1. Data Cleaning & Preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Feature Engineering  
4. Model Building  
5. Model Evaluation  

---

## 🤖 Machine Learning Models
- Logistic Regression  
- Support Vector Machine (SVM)

---

## 📈 Model Performance
- Best Model: Logistic Regression  
- ROC-AUC Score: 0.94  
- Balanced performance in classification metrics (Precision / Recall / F1-score)

---

## 🔑 Key Insights
- Performance score is the strongest predictor of promotion  
- Salary shows weak correlation with performance outcomes  
- Engineering department has the highest promotion rate  
- Workload indicators (overtime) highlight potential burnout risk
